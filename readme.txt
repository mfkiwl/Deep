1.norm(a)比sqrt(a*a')快
2.cos比cospi快,cospi比cos精确
3.修改一行或一列向量中一段的数值时,所提供的数不必特意转成行或者列
4.sscanf比str2num快
5.使用索引直接扩展向量长度时,默认是行
